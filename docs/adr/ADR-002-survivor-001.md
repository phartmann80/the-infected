# ADR-002 Survivor 001

## Status

Draft

## Approval Date

Pending

## Decision

Survivor 001 will be the first human concept and must establish the survivor visual language before any broader character roster is generated.

## Reasoning

One exceptional survivor is more valuable than ten average designs. Survivor 001 should make the world feel human, vulnerable, and worth surviving.

## Alternatives Considered

- Generate a squad of survivors immediately.
- Start with named heroes and backstories.
- Design multiple outfits before approving the core silhouette.

These alternatives create inconsistency before the visual language is established.

## Consequences

No additional survivor production should proceed until Survivor 001 has an approved brief, concept review, and registry-ready provenance.
