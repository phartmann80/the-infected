# ADR-004 Environment 001

## Status

Draft

## Approval Date

Pending

## Decision

Environment 001 will be the first approved environment concept and must answer: "What does The Infected look like?"

## Reasoning

The city is the emotional anchor of the project. Environment 001 should define the visual DNA for future locations through ruined urban streets, abandoned civilian life, emergency infrastructure, smoke, firelight, and cold shadows.

## Alternatives Considered

- Start with many locations at once.
- Start with generic zombie streets.
- Start with a gameplay arena before establishing world identity.

These alternatives risk creating content volume before identity.

## Consequences

Every later environment should inherit from Environment 001 unless a future accepted ADR expands or changes the location language.
