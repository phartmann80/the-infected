# Hero Production Candidate V1

This directory contains the reviewable plan for the first production-quality
hero cinematic. It is a prompt and shot manifest only; it intentionally contains
no generated image, video, or audio binaries.

The sequence follows the approved cinematic direction:

1. Establish the ruined city.
2. Reveal the checkpoint through fog and ash.
3. Build tension with emergency light and burning debris.
4. Suggest a survivor presence.
5. Surprise the viewer with distant infected movement.
6. Transition to the approved official logo as a web-owned overlay.
7. Hold the world before revealing the CTA.

Validate the package with:

```bash
npm run validate:hero-candidate
```

The manifest remains `canonical: false`, `reviewStatus: pending`, and
`generationMode: dry-run-only` until the creative review approves the concepts,
shot sequence, storage policy, and final browser composition.
